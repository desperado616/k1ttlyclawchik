# KittlyClaw - Студия маникюра и визажа премиум-класса

Современный, элегантный и конверсионный сайт для студии красоты с премиум-дизайном в пастельной цветовой гамме.

## Особенности

- Премиум-дизайн в пастельных тонах (розовый, бежевый, золотой)
- Полностью адаптивная верстка (мобильные, планшеты, десктоп)
- Элегантные SVG иконки вместо эмодзи
- Плавные анимации и переходы
- Онлайн-запись с формой бронирования
- Слайдер отзывов с автопрокруткой
- Таймер обратного отсчёта для акций
- Фильтрация портфолио
- Мобильная кнопка быстрого звонка
- SEO-оптимизация

## Структура сайта

1. **Header** - Логотип, навигация, контакты, кнопка записи
2. **Hero Section** - Главный экран с призывом к действию

5. **Мастера** - Карточки мастеров с рейтингом
6. **Портфолио** - Галерея работ с фильтрацией
7. **Отзывы** - Слайдер с отзывами клиентов
8. **Акции** - Специальные предложения с таймером
9. **Онлайн-запись** - Форма бронирования
10. **Контакты** - Информация и карта
11. **Footer** - Навигация, контакты, соцсети

## Рекомендации по изображениям

### Оптимальные размеры фотографий:

**Услуги (service-image):**
- Рекомендуемый размер: 800x600px (соотношение 4:3)
- Минимум: 640x480px
- Формат: JPG (качество 80-85%)

**Мастера (master-image):**
- Рекомендуемый размер: 600x800px (портрет, соотношение 3:4)
- Минимум: 480x640px
- Формат: JPG (качество 80-85%)
- Важно: Лицо должно быть в верхней части фото

**Портфолио (portfolio-image):**
- Рекомендуемый размер: 800x800px (квадрат 1:1)
- Минимум: 600x600px
- Формат: JPG (качество 80-85%)

### Оптимизация изображений:

Для сжатия изображений используйте:
- https://tinypng.com/ (онлайн)
- https://squoosh.app/ (онлайн, от Google)
- ImageOptim (Mac)
- FileOptimizer (Windows)

### Режимы отображения:

По умолчанию используется `background-size: cover` (изображение заполняет всю область, может обрезаться).

Если нужно показать изображение целиком без обрезки, добавьте класс `image-contain`:
```html
<div class="service-image image-contain" style="background-image: url('...');"></div>
```

## Настройка карты

Для работы Яндекс.Карт необходимо:

1. Получить API-ключ на https://developer.tech.yandex.ru/
2. В файле `index.html` заменить `YOUR_API_KEY` на ваш ключ:
   ```html
   <script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU"></script>
   ```
3. В файле `script.js` изменить координаты на адрес вашей студии:
   ```javascript
   const coordinates = [55.751244, 37.618423]; // [широта, долгота]
   ```

Чтобы узнать координаты вашего адреса:
- Откройте https://yandex.ru/maps
- Найдите нужный адрес
- Кликните правой кнопкой мыши на точку
- Выберите "Что здесь?" - координаты появятся внизу

## Установка и запуск

1. Скачайте все файлы проекта
2. Откройте `index.html` в браузере
3. Для локального сервера используйте:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js (с установленным http-server)
   npx http-server
   ```
